# 패키지와 접근제어 (package and access control)

## 패키지란?

자바에서도 프로그램을 작성하다 보면 많은 클래스와
인터페이스가 생기는데, 이를 폴더에 넣어 두는 것처럼 패키지에
담아 관리합니다.

사용법 : 클래스 선언 위에 package를 붙임
